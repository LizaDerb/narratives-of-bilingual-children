# narratives-of-bilingual-children
## Исследование речи детей-билингвов
Предметом исследования является анализ механизмов использования языка при построении связного нарратива на русском языке детьми-билингвами дошкольного, младшего, среднего и старшего школьного возраста 
Исследование внесёт вклад в изучение особенностей построения нарратива детьми-билингвами на русском языке, позволит выделить типичные для данной группы ошибки и может послужить основой для создания специализированного курса по изучению русского языка
## Дальнейшие перспективы исследования
Продолжение сбора и анализа нарративов
Расширение базы контактов
Выдвижение новых гипотез и их проверка
Разработка новых критериев для оценки ошибок
Поиск критериев сравнения
Поиск возможности проведения лонгитюдного исследования
